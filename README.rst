Install
------------------------------------------------------------------------------

``prompt_risk`` is released on PyPI, so all you need is to:

.. code-block:: console

    $ pip install prompt-risk

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade prompt-risk
